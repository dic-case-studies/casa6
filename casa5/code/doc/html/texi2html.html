Experimental AIPS++ use of texi2html

Recently the program "texi2html" was posted to the net. This offers the promise of much nicer online display of documents written in texinfo. As an experiment, Coding and Database have been converted using this program. This will probably be the standard for AIPS++ at some point. Note that these documents aren't presently being updated as the versions available through the infogate gateway are.

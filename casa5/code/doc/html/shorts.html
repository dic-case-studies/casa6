Guide for AIPS++ Help Writers

TermLaTeX Shortcut
AIPS++\aipspp
Glish\glish
AipsView\av
tool\tool
tools\tools
tool function\toolfunction
tool functions\toolfunctions
constructor\constructor
constructors\constructors

Note:

  • When defining new commands, please avoid using \newcommand and instead use \providecommand.

  • Also, to avoid checking in .help files with errors and waiting for the build to see if something is broken, please gmake yourfilename.dvi before your check in. This will catch any of those nasty errors that would otherwise popup in the build.

  • All definitions in use by AIPS++ can be found in code/doc/texinputs.dir/aips2defs.tex.

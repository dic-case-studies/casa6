Frequently Asked Questions About AIPS++

  • What is AIPS++?
    AIPS++ is the Astronomical Information Processing System.

  • How was it developed?
    • By a consortium of radio observatories. About 40 people in all contributed to the first release in 1999. The design was based on object-oriented techniques, and the implementation was in C++ and a glue language called Glish.

  • What does it do?
    We quote the mission statement:
    A telescope requires post-processing software for calibration, editing, image formation, image enhancement, and analysis of images and other data streams. This software is an integral part of the radio telescope engineering. The Astronomical Information Processing System (AIPS++) project is designed to produce such a software product. Although AIPS++ is primarily targeted at radio astronomy, it is anticipated that it will also be used in other branches of astronomy and for other applications in image processing and data analysis.

Defunct AIPS++ E-mail Reflectors

AIPS++ now uses the mailman archive facility to log email. Previously we used an mh program to archive email. These old messages maybe accessed using this page.

Archives of lists prior to Dec 2003:

  • aips2: General announcements and reports about AIPS++. This list presently has about one message a month.
  • aips2-beta: General announcements and reports about the AIPS++ beta release.
  • aips2-bugs: This is where bugs about AIPS++ should be reported.
  • aips2-plugins: A repository for AIPS++ plugins.
  • The following reflectors are likely to be of interest only those actively working on AIPS++:
    • aips2-astro-design: Discussion of issues related to the "high level" astronomical design.
    • aips2-changes: Automatically generated change log messages that accompany source code check-ins.
    • aips2-doc: Discussion concerning AIPS++ documentation.
    • aips2-dish: Discussion concerning AIPS++ dish package.
    • aips2-glish: Discussion relating to Glish.
    • aips2-inhale: The most recent logs of rebuilds of AIPS++ at the various consortium sites.
    • aips2-lib: Discussion of AIPS++ core library issues.
    • aips2-modulebosses: AIPS++ Module bosses' reports.
    • aips2-multibeam: The Parkes multibeam project.
    • aips2-parallel: Discussion of AIPS++ parallel issues.
    • aips2-pipelines: General discussion about pipelines using AIPS++.
    • aips2-reports: AIPS++ progress reports, see also aips2-weekly-reports.
    • aips2-sitemgr: Announcements and discussions intended for managers of AIPS++ installations.
    • aips2-synthesis: Discussions of the design, implementation and use of synthesis system.
    • aips2-system: Discussions of the code distribution, code management, and code compilation system.
    • aips2-testing: AIPS++ testing and related issues.
    • aips2-ui: Discussion of user interface and related issues.
    • aips2-visualization: Discussion of display library and visualization issues.
    • aips2-workers: A general-purpose reflector for people working on the AIPS++ project.
  • The following reflectors are likely to be of interest only to people at particular consortium sites:
    • aips2-atnf: Australia Telescope National Facility.
    • aips2-nfra: Netherlands Foundation for Research in Astronomy.
    • aips2-nrao: National Radio Astronomy Observatory.
      • aips2-aoc: Array Operations Center (Socorro, NM).
      • aips2-cv: Headquarters (Charlottesville, VA).
      • aips2-gbt: Green Bank Telescope (Green Bank, WV).
